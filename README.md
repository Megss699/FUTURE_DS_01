# E-Commerce Sales Analysis Dashboard (Power BI)

## Project Overview
This project involves analyzing e-commerce sales data to identify best-selling products,
sales trends, and high-revenue categories using Power BI.

## Objectives
- Identify top-performing products
- Analyze sales trends over time
- Determine high-revenue product categories
- Support data-driven business decisions

## Tools & Skills Used
- Power BI
- Data Cleaning
- DAX
- Trend Analysis
- Business Storytelling

## Files in This Repository
- Sample - Superstore.xls : Dataset used for analysis
- TASK 1 DASHBOARD.pbix : Interactive Power BI dashboard
